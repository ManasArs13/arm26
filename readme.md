


## Arm26.ru

Проект создан для армянской общины по Ставропольскому краю.

## В приложение возможно осуществить.
    1. пользователям.
       - авторизация
        - добавление услуг
        - поиск
    2. администраторам.
        -управлять категориями
        -управлять постами
        -управлять пользователями
    

## Использовались:
    - laravel 5.8
    - vue 2 (SPA для админ-панель)
